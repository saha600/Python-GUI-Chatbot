## Project Description

Python GUI Chatbot is a desktop chatbot application developed using Python and Tkinter. It provides a user-friendly graphical interface where users can interact with the chatbot through text messages. The chatbot can respond to greetings, provide the current date and time, tell jokes, and answer simple conversational queries.

---

## Features

- GUI using Tkinter
- Scrollable Chat Window
- Send Button
- Press Enter to Send Messages
- Clear Chat Button
- Exit Button
- Greeting Responses
- Date and Time Responses
- Random Jokes
- Friendly Conversation
- Easy-to-use Interface

---

## Technologies Used

- Python 3
- Tkinter
